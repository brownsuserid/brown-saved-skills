After archiving a batch, immediately pull the next batch's thread. Do NOT ask "want me to pull the thread?" or "should I pull the next one?"

**Why:** It's an unnecessary question that slows triage down. Josh always wants the thread pulled.

**How to apply:** Any time a batch is archived and there is a next batch to process, pull its thread immediately as part of the same action.
