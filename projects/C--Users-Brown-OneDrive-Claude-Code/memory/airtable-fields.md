# Airtable Field Reference

## Deals Table (`tblw6rTtN2QJCrOqf`)

### Editable Fields
- **Name** (text): Deal name
- **Description** (rich text): Deal description
- **Pain Points** (rich text): Lead's pain points
- **Assignee** (link to Employees): Deal owner
- **Status** (link to Pipeline Stages): Current stage
- **Organization** (link to Organizations): Company
- **Amount** (currency): Deal value
- **Probability** (percent): Win probability
- **Closing Date** (date): Expected close
- **Type** (select): New Business | Existing Business | Partner
- **Budget** (currency): Known budget
- **Engagement** (select): Low | Medium | High | On Hold
- **Tags 2** (select): Outreach tracking status
- **Deal Contacts** (link to Deal Contacts)
- **Campaigns** (link to Campaigns)
